# Median-of-Two-Sorted-Arrays
Given two sorted arrays nums1 and nums2 of size m and n respectively, return the median of the two sorted arrays.<br/>

Constraints:<br/>
  nums1.length == m<br/>
  nums2.length == n<br/>
  0 <= m <= 1000<br/>
  0 <= n <= 1000<br/>
  1 <= m + n <= 2000<br/>
  -106 <= nums1[i], nums2[i] <= 106
